# RECLAIM

RECLAIM is a robotics capstone project focused on autonomous navigation and task completion.

## Project Overview

This repository contains the codebase for the RECLAIM robotics system.

## Getting Started

### Prerequisites

- Python 3.8+
- ROS (Robot Operating System)
- Required Python packages (see requirements.txt)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd CapstoneProject

# Install dependencies
pip install -r requirements.txt
```
